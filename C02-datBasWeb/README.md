# DBWeb – Ejercicios de Conexión y Operación con Bases de Datos en la Web

Esta carpeta contiene ejercicios desarrollados para la asignatura **DBWeb**, cuyo foco principal es el diseño 
y la implementación de aplicaciones que interactúan con bases de datos alojadas remotamente.

Aquí se incluyen ejemplos prácticos que muestran:

- Conexión a bases de datos disponibles en servidores remotos.
- Operaciones CRUD básicas (Create, Read, Update, Delete).
- Uso de APIs o middleware para vincular aplicaciones web con sistemas de bases de datos.

Los ejercicios tienen fines demostrativos y académicos, buscando evidenciar la correcta operación y 
comunicación entre componentes distribuidos.
