# Ejercicios de Frontend

Este directorio contiene pruebas sencillas del lado del cliente.  
La intención es practicar con ejemplos básicos de HTML, CSS y JavaScript,  
centrándose en la claridad y la simplicidad.

---

## Propósito
- Explorar cómo se construyen interfaces web.
- Ensayar estilos y pequeñas interacciones.
- Mantener cada ejercicio lo más simple posible.

---

## Nota
Los ejemplos aquí no pretenden ser aplicaciones completas.  
Son ejercicios breves para reforzar conceptos fundamentales del frontend.
