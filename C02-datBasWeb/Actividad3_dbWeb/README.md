# Actividad 3 – Firebase Cloud Firestore (IoT)

Esta carpeta contiene una actividad realizada para la asignatura **DBWeb**.  
Es un trabajo individual, donde se conecta un microcontrolador (simulado) con Firebase 
Cloud Firestore para registrar y consultar datos en la nube.

En el notebook se documenta:

- Objetivo de la actividad.
- Configuración del proyecto en Firebase.
- Conexión desde el entorno de simulación / microcontrolador.
- Lectura y escritura en Cloud Firestore.
- Consultas y pruebas de funcionamiento.
- Capturas y evidencias.

**Todo el detalle se encuentra dentro del archivo `.ipynb` y en su exportación `.html`.**
