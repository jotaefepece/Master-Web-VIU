# Ejercicios y Proyectos de Estudios de Postgrado

Este repositorio contiene distintas actividades, ejercicios y prototipos desarrollados durante 
mis estudios de postgrado en área de Sistemas, Programación y Tecnologías web.

El objetivo principal de esta carpeta es **mostrar ejemplos reales de trabajo**, tanto a nivel 
de diseño como de implementación, que puedan servir como referencia para proyectos futuros, 
documentación técnica o portafolio profesional. 

Cada subcarpeta corresponde a una asignatura, módulo o línea temática específica.
