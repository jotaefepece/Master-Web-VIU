# Ejercicios de Backend

Este directorio reúne prácticas básicas del lado del servidor.  
El objetivo es experimentar con conceptos simples de programación backend y bases de datos,  
manteniendo los ejemplos claros y fáciles de revisar.

---

## Propósito
- Probar ideas sin complejidad innecesaria.
- Familiarizarse con la lógica del servidor.
- Documentar cada ejercicio de forma breve.

---

## Nota
Estos ejercicios son de carácter exploratorio.  
No buscan ser proyectos completos, sino pequeños pasos para afianzar conceptos.
